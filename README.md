# OO-muti-agent-systems
Object-oriented simulation of multi-agent systems

### Compilation
`make`

### Execution

Run Ball Simulation
`make exeBalls`

Run Game of Life Simulation
`make exeVie`

Run Imigration Simulation
`make exeImi`

Run Schelling Simulation
`make exeSeg`

Run Boids Simulation
`make exeBoids`
